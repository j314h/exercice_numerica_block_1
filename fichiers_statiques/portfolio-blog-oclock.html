Blog-Oclock

Voir le site

J'avais dans l'esprit de créer un blog pour écrire mes différentes péripéties pendant ma formation chez Oclock. Se fut également mon premier projet avec un back-end créé avec ExpressJS. Partie front-end, j'ai utilisé MatérialUI. J'ai pu faire de nombreux tests et progresser dans différentes technologies que je ne connaissais pas du tout avant de me lancer dans le projet. Malheureusement, j'ai dû abandonner le projet, pour non-divulgation du programme de formation chez Oclock. J'ai hébergé le front-end avec Github et le back-end chez Héroku, aucune base de données est reliée au site, il n'y a donc aucun article sinon ceux enregistrés initialement.
